## üïπÔ∏è Key Features & Mechanics

### Core Gameplay Loop

  * **Memory Recovery:** The King must locate and interact with the magical Memory Trees to reclaim Memory Fragments.
  * **Ability Progression:** Claiming Memory Shards unlocks new magical abilities to aid in combat and traversal.
  * **Final Choice:** Upon defeating The Keeper, the player determines his fate by choosing to **Spare or Execute** him.

### Player Abilities (King Aldric)

The player utilizes both sword combat and magical skills:

| Ability | Type | Description |
| :--- | :--- | :--- |
| **Jump** | Movement | Overcomes obstacles. |
| **Double Jump** | Movement | Allows the King to reach greater heights while mid-air. |
| **Dash** | Movement | A swift leap forward used to dodge attacks and avoid hazards. |
| **Life Siphon (Lvl 1)** | Magic / Passive | The King heals when defeating an enemy (with a cooldown). |
| **Liquid Shield (Lvl 2)** | Magic / Defense | Creates a water bubble that can tank one attack. |
| **Fire Ball (Lvl 3)** | Magic / Ranged | The King casts a ball of fire towards his target. |

### Dynamic Challenges

The King must traverse four distinct, hostile realms, each with unique environmental physics and enemies:

  * **The Withering Forest:** Features acid floors, prickly vines, bouncing mushrooms, and enemies like the Bush Creep and Snapmaw.
  * **Ice Realm:** Challenges include slippery ice physics, fragile platforms, and dangerous enemies like the Ice Golem and Frost King.
  * **Volcano/Fire Realm:** Players must dodge falling spikes and rising lava, battling fire spirits and dragons (Volcarion).
  * **The Castle:** The final level involves solving a magical gate puzzle, avoiding dungeon traps (trapdoor, saw blades), and facing the final boss, The Keeper.
